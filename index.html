How to Use the Percentage Calculator

Calculator Modes

  • What is X% of Y:
    Example: What is 15% of 200 = 30
    • Enter the percentage (X) in the first field
    • Enter the number (Y) in the second field
    • Result shows instantly as you type
  • X is what % of Y:
    Example: 50 is what % of 200 = 25%
    • Enter the number (X) in the first field
    • Enter the total (Y) in the second field
    • Result shows instantly as you type
  • % change from Origin to New:
    Example: Change from 100 to 150 = 50% increase
    • Enter the original value in the first field
    • Enter the new value in the second field
    • Shows the percentage increase/decrease

Tips

  • All calculations update in real-time as you type
  • Use the Reset button to clear all inputs
  • Results are rounded to 2 decimal places
  • Negative percentages are supported
