How to Use the Scientific Calculator

Basic Operations

  • Arithmetic:
    • Addition (+)
    • Subtraction (-)
    • Multiplication (×)
    • Division (÷)
  • Numbers: Use the number pad (0-9) and decimal point
  • Equals (=): Calculate the result
  • Clear (C): Reset the calculator

Scientific Functions

  • Trigonometric:
    • sin: Sine (in degrees)
    • cos: Cosine (in degrees)
    • tan: Tangent (in degrees)
  • Logarithmic:
    • log: Base-10 logarithm
    • ln: Natural logarithm
  • Powers:
    • x²: Square
    • x³: Cube
    • xʸ: Power
    • √: Square root
  • Other:
    • n!: Factorial
    • 1/x: Reciprocal
    • %: Percentage

Memory Functions

  • MC: Memory Clear - Clears the stored value
  • MR: Memory Recall - Shows the stored value
  • M+: Memory Add - Adds current value to memory
  • M-: Memory Subtract - Subtracts current value from memory

Tips

  • Use keyboard for basic operations and numbers
  • Expression shows at the top of the display
  • Result shows below the expression
  • Press ESC key or C button to clear
  • Scientific functions work on the current displayed number
